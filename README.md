# WinSpotLight Viewer

专注于查看和提取Windows聚焦图片。

暂未完工，基本功能可用。

## Features

基本特性：
- 能链接，部分软件直接就能正确查看图片，如: VS Code。
- 能提取。

更多特性：
- 支持对已提取的链接。
- 支持相对路径链接。

## Problems to Solve

- [ ] 来源是内容交付机制的目录，其中可能混杂有磁贴。
- [ ] 来源文件并非总是用相对路径更好，都优先尝试相对路径链接会导致程序运行缓慢。
- [ ] 提取得到的和过往的混杂在一起，链接到的也是，这不利于查看新增，不利于管理。
- [ ] 链接得到的并非所有软件都能很好识别和查看，如资源管理器无法预览和用预设打开。
